Stay In Touch with xCAT developers

  • xCAT Community in GitHub : xCAT core code repository and issues, xCAT users can contribute code, file bug report or raise new feature request here.

  • xcat-user mail list on SourceForge : A mail list with the majority of xCAT users to discuss any xCAT related questions. The threads can be archived and reachable from Google

  • Gitter : A IM channel among xCAT developers and xCAT users on any xCAT related topics, it is more realtime compared with the xcat-user mail list. The threads cannot be reachable from Google
